# Buscaminas

# ¿Cómo jugar?

1. Ingresá el tamaño del tablero (entre 6 y 20).
2. Ingresá la cantidad de bombas.
3. Hacé clic en "Generar Juego".
4. Click izquierdo --> para revelar una casilla.
5. Click derecho --> para colocar o quitar una bandera.
6. Doble click --> sobre una casilla marcada revela las casillas adyacentes si ya están marcadas todas las bombas alrededor.

# Cómo ejecutar

1. Cloná el repositorio o descargá los archivos.
2. Abrí `index.html` con cualquier navegador web.
3. ¡A jugar!
